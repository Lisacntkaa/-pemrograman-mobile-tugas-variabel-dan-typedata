Deskripsi
Program ini dibuat menggunakan bahasa Dart untuk menghitung **gaji bersih karyawan**. Perhitungan didasarkan pada jam kerja per minggu, Upah per jam dan Status karyawan (tetap/kontrak)

Potongan pajak:
- Karyawan tetap : 10%
- Karyawan kontrak : 5%
